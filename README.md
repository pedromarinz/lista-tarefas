# Lista de Tarefas

Este projeto é uma aplicação de gerenciamento de tarefas desenvolvida para consolidar conhecimentos em **Manipulação Dinâmica do DOM** e **Gestão de Estados** no Front-end. A ferramenta permite organizar o fluxo de trabalho diário de forma simples e intuitiva.

## 🚀 Funcionalidades

* **Adição de Tarefas:** Interface rápida para criação de novos itens com suporte à tecla `Enter`.
* **Marcação de Conclusão:** Sistema de "toggle" que destaca tarefas finalizadas e as move automaticamente para o fim da lista.
* **Exclusão Individual:** Gerenciamento de remoção de itens via IDs únicos.
* **UX/UI Moderna:** Design responsivo com feedback visual de interação (`hover`, `focus` e `active`).
* **Ícones Intuitivos:** Integração com a biblioteca FontAwesome para melhor navegação.

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação semântica da página.
* **CSS3:** Layout baseado em **Flexbox** e variáveis de cores para fácil manutenção.
* **JavaScript (ES6+):** Lógica de controle, criação de elementos dinâmicos via Template Literals e gerenciamento de eventos.
* **FontAwesome:** Provedor de ícones vetoriais.
